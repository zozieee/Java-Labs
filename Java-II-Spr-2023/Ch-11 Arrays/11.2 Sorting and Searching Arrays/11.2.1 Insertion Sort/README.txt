Given an array of integers, sort the array in ascending/descending
order by using Insertion sort.
Reference: http://www.algolist.net/Algorithms/Sorting/Insertion_sort

Input
0
9
4 3 6 8 9 2 1 5 7

Where,
• First line represents the type of ordering.
• Second line represents the size of the array.
• Third line represents the array elements.

Output
1 2 3 4 5 6 7 8 9
